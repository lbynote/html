# 什么是HTML
HTML是用来描述网页的一种语言。
- HTML指的是超文本标记语言(Hyper Text Markup Language)。
- HTML不是一种编程语言，而是一种标记语言(markup language)。
- 标记语言是一套标记标签(markup tag)。
- HTML使用标记标签来描述网页。

# HTML标签  
HTML标记标签通常称为HTML标签(HTML tag)
- HTML标签是由尖括号包围的关键词，比如<html>。
- 标签对中的第一个标签是开始标签/开放标签，第二个标签是结束标签/闭合标签。

# HTML文档 = 网页
- HTML文档描述网页。
- HTML文档包含HTML标签和文本。
- HTML文档也被称为网页。
